# CylinderMeshingTool
A high-fidelity, free user input cylinder meshing tool for MCNP, which can be used for neutronics and thermal hydraulic coupling.

This code is used in "Zhou, X. and Zhan, D*. and Wang, X. et al. *A coupling analysis method of the thermal hydraulics and neutronics based on inverse distance weighted method*. NURETH-19, Brussel, 2022"
